# E-commerce-website-
Luxury-themed e-commerce website with elegant typography, gold accents, and a modern layout. Features full-screen hero, product listings with add-to-cart functionality, interactive cart, contact form, and fully responsive design. Easy to customize images, banners, logos, products, prices, and branding.

Features

Full-screen hero section with luxury banner

Product listings with images, titles, prices, and "Add to Cart" buttons

Interactive shopping cart with total calculation and remove item functionality

Contact form for customer messages

Fully responsive and mobile-friendly

Easy to customize: replace images, banners, logos, product names, and prices
